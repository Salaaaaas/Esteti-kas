# Esteti-kas

**Esteti-kas** is the repository for the *Esteti'kas* website.

## Overview
This project contains the source code and assets for the Esteti'kas web presence. It is intended to be a simple, maintainable front-end codebase.

## Tech Stack
GitHub currently detects **CSS** as the primary language for this repository. Depending on the final site structure, the project may also include:
- **HTML** for markup
- **JavaScript** for interactivity


## Getting Started
Because this is a website repository, you can typically preview it locally by opening the main HTML file (e.g., `index.html`) in your browser, or by serving the folder with a simple static server.

### Option A: Open directly
1. Clone the repository.
2. Open the entry file (commonly `index.html`) in your browser.

### Option B: Serve locally (recommended)
Use any static server, for example:
- VS Code extension: **Live Server**
- Node.js: `npx serve` (if you have Node installed)

## Project Structure
The structure may evolve as the website grows. A typical layout looks like:
- `index.html`
- `assets/` (images, icons, fonts)
- `css/` (stylesheets)
- `js/` (scripts)
